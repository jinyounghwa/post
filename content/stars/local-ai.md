## exo-explore/exo
- **링크:** https://github.com/exo-explore/exo
- **설명:** Run frontier AI locally
- **왜 좋냐:** 집에 있는 맥, 아이패드, 아이폰을 묶어서 프론티어 모델 로컬 실행 가능. 43,000+ 스타. 진짜 됨

## lyogavin/airllm
- **링크:** https://github.com/lyogavin/airllm
- **설명:** 70B inference with a single 4GB GPU
- **왜 좋냐:** 4GB VRAM으로 70B 모델 추론. 양자화 없이 레이어 오프로딩으로 구현. 14,400+ 스타

## maderix/ANE
- **링크:** https://github.com/maderix/ANE
- **설명:** Train neural networks on Apple Neural Engine via reverse-engineered private APIs
- **왜 좋냐:** Apple Neural Engine의 비공개 API를 역분석해서 직접 학습에 활용. 6,500+ 스타. 실험적이지만 흥미로움

## memovai/mimiclaw
- **링크:** https://github.com/memovai/mimiclaw
- **설명:** Run OpenClaw on a $5 chip — no OS/Node.js/Mac required
- **왜 좋냐:** 5달러짜리 칩에서 OpenClaw 실행. IoT·엣지 AI의 극한. OS도 Node.js도 필요 없음

## localgpt-app/localgpt
- **링크:** https://github.com/localgpt-app/localgpt
- **설명:** Local AI assistant, dreaming explorable worlds
- **왜 좋냐:** Rust 기반 로컬 AI 어시스턴트. 프라이버시 중시하는 사용자에게 적합

## Tencent/AngelSlim
- **링크:** https://github.com/Tencent/AngelSlim
- **설명:** Model compression toolkit for usability and efficiency
- **왜 좋냐:** 텐센트가 만든 모델 압축 툴킷. 양자화, 프루닝, 증류를 한 번에. 로컬 배포 최적화 필수템

# CLAUDE.md — jinslife.kr 포트폴리오 전면개편

## 0. 한 줄 요약
현재 "AI 뉴스 큐레이션 허브"로 되어 있는 jinslife.kr(레포: jinyounghwa/post)를,
**"기획자, 개발을 약간하는 진영화"**의 영업/포트폴리오 사이트로 전면개편한다.
뉴스/Stars 큐레이션 기능은 죽이지 않고 서브 메뉴로 내린다.

---

## 1. 목표 (Why)

- 방문자가 30초 안에 "이 사람은 기획도 하고 개발도 직접 하는, 실제로 뭘 만들 수 있는 사람"이라는 인상을 받게 한다.
- 말로 된 경력 소개가 아니라 **실제 동작하는 데모 8건**을 신뢰 근거로 제시해 설득력을 확보한다.
- 방문 → 데모 시청/체험 → Contact 전환이 이어지는 도선(導線)을 만든다.
- 기존 뉴스 큐레이션(콘텐츠 자산)은 "이 사람은 트렌드도 놓치지 않는다"는 신뢰 보강 요소로 재배치한다.

## 2. 사용자 (Who)

| 우선순위 | 대상 | 방문 목적 |
|---|---|---|
| 1 | 스타트업 대표 / PM | MVP 외주 개발자 탐색, "기획+개발 동시 가능"이 핵심 셀링포인트 |
| 2 | 사내 담당자 (쇼핑몰/물류/공공기관) | 기존 서비스 리뉴얼, 특정 도메인(WMS, 커머스) 경험 확인 |
| 3 | 에이전시/기획사 | 협업 파트너로서의 개발 역량 검증 |
| 4 | 채용 담당자 | 이력 검증용 방문 (부차적) |

### 3.2 Portfolio — 데모 8종 (이번 개편의 핵심)

기존 README에 있던 옛 포트폴리오 항목(AI 댓글 자동화 등, Facebook Reels 임베드)은 **이번 8개로 교체**한다.

| # | 프로젝트 | 미디어 | 카테고리 | 강조 포인트 |
|---|---|---|---|---|
| 1 | 쇼핑몰 데모 (쿠폰 로직+관리자) | YouTube | E-commerce | 쿠폰엔진, 주문/배송 상태머신, 관리자 대시보드 — WMS 도메인 지식이 녹아든 물류 시뮬레이션 포함 |
| 2 | OpenTracker | YouTube | Analytics | 웹 방문 페이지 행동 분석 툴 자체 구현 |
| 3 | Antigravity Skills 제품구매 AI분석 | YouTube | AI 활용 커머스 | AI 스킬을 활용한 구매 사이트 분석 자동화 |
| 4 | 공공데이터 연동 차트 (NestJS) | YouTube | 백엔드/데이터 시각화 | NestJS API 게이트웨이 + 공공데이터 연동 + 차트 구현 |
| 5 | 로컬 AI 개발공부 보조 챗봇 | YouTube | AI/LLM | 로컬 LLM(Ollama) 기반, 비용 없는 학습 보조 챗봇 |

| 7 | SoundScape | 라이브 데모(Netlify) | 프론트엔드/감성 UX | ASMR 웹서비스, 인터랙션·오디오 UX 구현력 |

각 항목은 카드형 UI로: 썸네일/임베드, 한 줄 설명, 기술스택 태그, 역할, "데모 보기" 외부링크.
상세 스펙과 카피 문구는 SKILL.md 3장 참조.

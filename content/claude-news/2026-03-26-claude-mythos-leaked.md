## Mythos, 베일을 벗다

데이터 유출로 먼저 세상에 알려진 **Claude Mythos**를 Anthropic이 공식 확인했다. 내부 문서에서는 *"우리가 개발한 것 중 단연 가장 강력한 AI 모델"*이라고 표현했으며, 얼리 액세스 고객을 대상으로 테스트가 진행 중이다.

### 알려진 사양

- **Capybara 에디션** — 프로그래밍 태스크와 추론에서 Claude Opus 4.6 대비 "극적인" 성능 향상
- **고급 추론 엔진** — 복잡한 다단계 문제에서 체계적 추론 강화
- **코딩 특화** — Claude Code 차세대 백엔드로 활용 예정

### 왜 주목해야 하나

Mythos가 정식 출시되면 Claude Code 및 에이전트 워크플로우의 핵심 엔진이 교체된다. 특히 **대규모 코드베이스 분석**과 **멀티스텝 에이전트 태스크**에서 체감 성능 차이가 클 것으로 예상된다.

### 현재 상태

```
[테스트 중] → 얼리 액세스 고객
[미정]       → 공개 출시 일정
```

Fortune, SiliconANGLE 등 주요 매체가 일제히 보도하며 업계의 관심이 집중되고 있다.

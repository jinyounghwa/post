## jj-vcs/jj
- **링크:** https://github.com/jj-vcs/jj
- **설명:** A Git-compatible VCS that is both simple and powerful
- **왜 좋냐:** Git 호환 VCS인데 개념이 훨씬 단순함. 브랜치 없이 작업 가능. Rust 기반. 27,300+ 스타

## manaflow-ai/cmux
- **링크:** https://github.com/manaflow-ai/cmux
- **설명:** Ghostty-based macOS terminal with vertical tabs and notifications for AI agents
- **왜 좋냐:** AI 에이전트 전용 터미널. 세로 탭 + 알림 지원. Claude Code 같은 에이전트와 찰떡 궁합. 10,900+ 스타

## steipete/CodexBar
- **링크:** https://github.com/steipete/CodexBar
- **설명:** Show usage stats for OpenAI Codex and Claude Code, no login needed
- **왜 좋냐:** 메뉴바에서 Claude Code · Codex 사용량 실시간 확인. 로그인 불필요. 9,400+ 스타. 맥 유저 필수

## microsoft/playwright-cli
- **링크:** https://github.com/microsoft/playwright-cli
- **설명:** CLI for Playwright actions: record, generate code, inspect selectors, screenshots
- **왜 좋냐:** E2E 테스트 코드 자동 생성. 브라우저 조작 녹화 → 코드 변환이 에이전트 자동화와 시너지

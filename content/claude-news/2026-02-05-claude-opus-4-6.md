## Claude Opus 4.6 정식 출시

Anthropic이 **Claude Opus 4.6**을 출시했다. 가장 눈에 띄는 변화는 두 가지다.

### 핵심 스펙

| 항목 | 수치 |
|------|------|
| 컨텍스트 윈도우 | **1,000,000 토큰** |
| 최대 작업 시간 | **14.5시간** |
| 입력 가격 | $5 / 1M 토큰 |
| 출력 가격 | $25 / 1M 토큰 |

### 1M 컨텍스트가 개발에 미치는 영향

기존에는 대규모 코드베이스를 분석할 때 파일을 나눠서 전달하거나 별도 RAG 파이프라인을 구성해야 했다. 1M 토큰이면 **수백 개의 소스 파일을 한 번에** API에 넣을 수 있다.

```
기존: 파일 청킹 → 벡터 검색 → 컨텍스트 조립
이제: 전체 레포 → Claude API → 분석 완료
```

### 14.5시간 작업 호라이즌

단순 Q&A가 아니라 **장시간 자율 에이전트** 운용이 가능해졌다. 복잡한 마이그레이션, 전체 기능 구현, 대규모 리팩토링을 사람 개입 없이 이어서 처리할 수 있다.

### 멀티 에이전트 팀 기능

Claude Opus 4.6은 Anthropic의 새로운 **멀티 에이전트 "팀"** 기능의 핵심 엔진이다. 여러 Claude 인스턴스가 협력해 복잡한 태스크를 병렬로 처리한다.

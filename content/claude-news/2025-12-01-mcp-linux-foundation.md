## MCP, Anthropic의 손을 떠나다

Anthropic이 **Model Context Protocol(MCP)** 를 리눅스 재단 산하 **Agentic AI Foundation(AAIF)** 에 기증했다. Anthropic, Block, OpenAI가 공동 창립했고, Google, Microsoft, AWS, Cloudflare, Bloomberg가 지원에 합류했다.

### 2025년 MCP의 성장

```
월간 SDK 다운로드:  9,700만 건
활성 MCP 서버:      10,000개 이상
지원 플랫폼:        사실상 모든 주요 AI 플랫폼
```

### November 2025 스펙 업데이트 주요 내용

- **비동기 작업** — 장시간 실행 태스크 처리
- **무상태(stateless) 모드** — 서버리스 환경 지원
- **서버 ID 검증** — 보안 강화
- **공식 커뮤니티 레지스트리** — 검증된 서버 목록 제공

Anthropic과 OpenAI는 **MCP Apps Extension(SEP-1865)** 을 공동 발표하며 경쟁사 간 협력이라는 이례적인 행보를 보였다.

### 개발자에게 의미하는 것

MCP가 특정 기업의 스펙이 아닌 **Linux Foundation 관리 오픈 표준**이 됨으로써, 장기적인 안정성이 보장됐다. 지금부터 MCP 서버를 구축하면 플랫폼 락인 없이 모든 AI 에이전트와 호환된다.

> GitHub, Slack, PostgreSQL — 이미 10,000개 이상의 서버가 오픈소스로 공개되어 있다.
